# 个人工作台

## 项目概述

个人资料和工作文件管理仓库，包含：
- **税务政策管理**：中国税务政策追踪和学习闭环
- **旅行规划**：2026年法意之旅签证和行程文件

## 目录结构

```
.
├── 工作/
│   └── 政策/
│       ├── 税务政策管理表.xlsx          # 主管理表（税务政策库）
│       ├── 税务政策管理表_模板.csv       # CSV 模板备份
│       └── <政策名称>/                  # 按政策名称命名的文件夹
│           ├── <政策名称>.doc           # 政策原文（HTML 格式，来自国家税务总局）
│           └── <附件名称>.pdf           # 政策附件
├── 2026十一法意之旅/                    # 旅行规划文件
└── .gitignore
```

## 税务政策管理工作流

### 添加新政策

1. 在 `工作/政策/` 下创建以政策标题命名的文件夹
2. 将政策原文（.doc）和附件（.pdf）放入该文件夹
3. 在 `工作/政策/税务政策管理表.xlsx` 的 **税务政策库** sheet 中新增一行
4. 填写所有字段，状态初始设为「待学习」
5. 阅读政策全文后更新学习日志、待确认事项和状态

### 表格字段说明

| 字段 | 说明 |
|------|------|
| 序号 | 自增编号 |
| 税种 | 下拉选择（增值税/企业所得税/消费税/关税/印花税等 16 种） |
| 文号 | 官方文号，如「财关税〔2026〕1号」 |
| 政策标题 | 完整政策名称 |
| 发布日期 | 格式 yyyy-mm-dd |
| 生效日期 | 格式 yyyy-mm-dd |
| 核心变化 | ≤50 字概括政策要点 |
| 对本岗位影响 | 结合企业实际判断 |
| 待确认事项 | 需进一步核实的问题 |
| 学习日志 | 格式：YYYY/M/D：内容 |
| 状态 | 待学习 → 学习中 → 已学习 → 待确认 → 已确认 → 已归档 |
| 最近更新 | 格式 yyyy-mm-dd |

### 状态流转

```
待学习 → 学习中 → 已学习 → 待确认 → 已确认
                                    ↘ 已归档
```

### 政策来源

- 国家税务总局法规库：https://fgk.chinatax.gov.cn/
- .doc 文件实质是 MIME HTML 格式，用 `textutil` 可提取文本
- .pdf 附件可用 `pypdf` 库读取
